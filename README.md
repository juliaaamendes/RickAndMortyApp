# RickAndMortyApp

Uma aplicação JavaScript que consome a API de Rick and Morty para listar personagens, exibir detalhes e permitir buscas e filtros.

## Demonstração

Clique na imagem para abrir o vídeo de demonstração no YouTube:

[![Demonstração do App](https://img.youtube.com/vi/xrVGnlYWTAk/maxresdefault.jpg)](https://www.youtube.com/shorts/xrVGnlYWTAk)

## Funcionalidades

- Listagem de personagens
- Busca por nome
- Filtros por espécie, status e gênero
- Página de detalhes do personagem com episódios relacionados
- Paginação/infinite scroll

- JavaScript

## Instalação

Pré-requisitos: Node.js e npm ou yarn.

1. Clone o repositório:
   ```bash
   git clone https://github.com/juliaaamendes/RickAndMortyApp.git
2. Entre na pasta do projeto:
   ```bash
   cd RickAndMortyApp 
3. Instale as dependências:
    ```bash
    npm install
    # ou
    yarn install
4. Rode em modo de desenvolvimento:
   ```bash
   npm start
   # ou
   yarn start
